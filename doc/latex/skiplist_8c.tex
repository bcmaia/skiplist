{\footnotesize\ttfamily status\+\_\+t skiplist\+\_\+update (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{skiplist_8h_abdc97ffb081c6c8994f3eead70d2978a}{Skip\+List}} $\ast$}]{skiplist,  }\item[{\mbox{\hyperlink{item_8h_aa3baf16dec9854a9f37f7b1e1e4bd086}{Item}} $\ast$}]{item }\end{DoxyParamCaption})}



Updates the value of an item in a skiplist. 

Searches an skiplist for an item and updates it to a new value if it was found.


\begin{DoxyParams}{Parameters}
{\em skiplist} & a ptr to the skiplist. \\
\hline
{\em item} & the item we are searching and its new value. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
status\+\_\+t {\ttfamily SUCCESS} if the item was found and updated to a new value, {\ttfamily NUL\+\_\+\+ERR} if any critial pointer passed to the function is NULL, {\ttfamily ALLOC\+\_\+\+ERR} if there was an issue allocating memory during the execution of the program, {\ttfamily NOT\+\_\+\+FOUND\+\_\+\+ERR} if the item is not found on the skiplist, {\ttfamily ERROR} if an unidentified error ocurred during the trace and search phase.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
Keep in mind an item has a key, which is used to identify it, and a value. The item you pass as a parameter should have a key to an item already on the skiplist and a new value. 
\end{DoxyNote}
